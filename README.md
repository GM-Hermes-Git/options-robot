# Options Robot — Торговый робот для опционов MOEX

Автоматизированный робот для торговли опционами на срочном рынке Московской биржи (MOEX) через API брокера Alor.

## Возможности (по этапам разработки)

| Этап | Функциональность | Статус |
|------|-----------------|--------|
| 1. Фундамент | Подключение (MOEX ISS / Alor API), Event Bus, базовый GUI | 🚧 В разработке |
| 2. Ядро стратегий | StrategyManager, TriggerEngine, OrderManager | ⏳ План |
| 3. Греки и P&L | Модель Блэка-76, расчёт теор. цены, P&L позиции | ⏳ План |
| 4. Хеджер и риски | DeltaHedger, RiskManager (SL/TP) | ⏳ План |
| 5. GUI (полный) | Все вкладки, редактор стратегий, мониторинг | ⏳ План |
| 6. Уведомления | Telegram-уведомления о событиях | ⏳ План |
| 7. Тестирование | Полное тестирование в демо-режиме | ⏳ План |
| 8. Сборка | PyInstaller .exe для Windows, документация | ⏳ План |

## Архитектура

Модульная архитектура с внутренней шиной событий (Event Bus).

```
core/                  # Бизнес-логика
  providers/           # Провайдеры данных и ордеров
    moex_provider.py   # MOEX ISS API (рыночные данные)
    alor_provider.py   # Alor Open API (рыночные данные + ордера)
    simulated_orders.py # Симулятор ордеров (отладка)
  event_bus.py         # Шина событий
  strategy_manager.py  # Управление стратегиями
  trigger_engine.py    # Триггеры по цене БА
  order_manager.py     # Выставление/снятие ордеров
  delta_hedger.py      # Дельта-хеджирование
  greeks_engine.py     # Расчёт греков (Блэк-76)
  risk_manager.py      # SL/TP, контроль рисков
gui/                   # Интерфейс (PyQt6)
database/              # SQLite: сделки, позиции
notifications/         # Telegram-уведомления
utils/                 # Логирование, конфигурация
```

## Установка и запуск

```bash
pip install -r requirements.txt
python main.py
```

## Платформа

- Разработка: Python 3.11+, кроссплатформенно (Linux/Mac/Windows)
- Целевая платформа: Windows 10/11 (.exe через PyInstaller)

## Безопасность

- Токены хранятся в зашифрованном виде (Fernet)
- Конфигурация с токенами исключена из Git (.gitignore)
